# Privacy Policy

This app processes photos and videos **on-device only**. No media files or metadata are uploaded to external servers.

## Data collected
- None. The app does not collect or transmit user data.

## Permissions used
- Media access: to scan photos/videos stored on the device
- Media location: to read embedded GPS metadata in photos/videos (if available)

## Data storage
- Local-only database used to index media and tags
- Users can delete app data at any time from Android settings
